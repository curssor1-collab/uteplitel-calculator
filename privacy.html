Политика конфиденциальности

Последняя редакция: 1 января 2025 г.

1. Общие положения

Настоящая Политика обработки персональных данных (далее — Политика) разработана в соответствии с Федеральным законом от 27.07.2006 № 152-ФЗ «О персональных данных» (далее — ФЗ-152) и определяет порядок обработки персональных данных пользователей сайта «Теплорасчёт РФ» (далее — Сайт).

Оператор: Администрация сайта «Теплорасчёт РФ» (далее — Оператор).

Сайт: интернет-ресурс, расположенный по адресу: https://curssor1-collab.github.io/uteplitel-calculator/, предоставляющий пользователям возможность выполнять расчёт толщины утеплителя в ознакомительных целях.

Используя Сайт, Пользователь выражает свое безоговорочное согласие с условиями настоящей Политики. В случае несогласия с условиями Политики Пользователь обязан прекратить использование Сайта.

2. Какие персональные данные собираются

Оператор может собирать следующие категории данных:

  • IP-адрес — автоматически передаётся устройством Пользователя при посещении Сайта. Используется для определения региона и защиты от атак.
  • Файлы cookie — небольшие текстовые файлы, сохраняемые браузером Пользователя. Необходимы для корректной работы Сайта и сбора анонимной статистики.
  • Данные Яндекс.Метрики — анонимные статистические данные о посещениях (источник трафика, поведение на Сайте, технические характеристики устройства). Не позволяют идентифицировать конкретного Пользователя.
  • Данные, введённые в калькулятор — выбранный регион, материалы, толщина стены. Эти данные обрабатываются исключительно на устройстве Пользователя с помощью JavaScript и не передаются и не сохраняются на сервере Оператора.
  • Данные формы обратной связи — имя, адрес электронной почты, текст сообщения. Предоставляются Пользователем добровольно при отправке сообщения через форму обратной связи.

3. Цели обработки персональных данных

Оператор обрабатывает персональные данные Пользователя в следующих целях:

  • Обеспечение функционирования Сайта и его корректной работы на устройстве Пользователя;
  • Сбор анонимной статистики посещений для улучшения качества работы Сайта;
  • Показ релевантной рекламы через Рекламную сеть Яндекса (РСЯ);
  • Ответ на обращения Пользователя, направленные через форму обратной связи;
  • Соблюдение требований законодательства Российской Федерации.

4. Правовые основания обработки

Обработка персональных данных осуществляется на основании:

  • Федерального закона от 27.07.2006 № 152-ФЗ «О персональных данных» (ст. 6);
  • Федерального закона от 13.03.2006 № 38-ФЗ «О рекламе»;
  • Согласия Пользователя на обработку персональных данных (выражается в виде продолжения использования Сайта).

5. Порядок обработки персональных данных

Обработка персональных данных осуществляется с использованием средств автоматизации. Оператор принимает все необходимые правовые, организационные и технические меры для защиты персональных данных от неправомерного или случайного доступа, уничтожения, изменения, блокирования, копирования, распространения, а также от иных неправомерных действий.

Сайт использует протокол HTTPS для шифрования данных, передаваемых между устройством Пользователя и сервером.

6. Права Пользователя

Пользователь имеет право:

  • Получить информацию, касающуюся обработки его персональных данных;
  • Требовать уточнения, блокирования или уничтожения своих персональных данных в случае, если они являются неполными, устаревшими, недостоверными, незаконно полученными или не являются необходимыми для заявленной цели обработки;
  • Отозвать согласие на обработку персональных данных (путём очистки cookie в браузере);
  • Обжаловать действия или бездействие Оператора в уполномоченном органе по защите прав субъектов персональных данных (Роскомнадзор).

7. Сроки обработки и хранения

Персональные данные обрабатываются и хранятся до момента прекращения использования Сайта Пользователем либо до момента отзыва согласия на обработку персональных данных. Файлы cookie хранятся на устройстве Пользователя в течение сроков, установленных браузером. Пользователь может самостоятельно удалить cookie в настройках своего браузера.

8. Передача данных третьим лицам

Оператор передаёт персональные данные третьим лицам только в следующих случаях:

  • Яндекс.Метрика — для сбора анонимной статистики посещений. Передаются только обезличенные данные.
  • Рекламная сеть Яндекса (РСЯ) — для показа рекламы. Передаются только обезличенные данные.
  • По требованию законодательства — в случаях, предусмотренных законодательством Российской Федерации.

Оператор не продаёт, не сдаёт в аренду и не передаёт иным способом персональные данные Пользователя третьим лицам без его согласия, за исключением случаев, указанных выше.

9. Защита персональных данных

Оператор принимает следующие меры для защиты персональных данных Пользователя:

  • Назначение ответственного за организацию обработки персональных данных;
  • Применение правовых, организационных и технических мер обеспечения безопасности персональных данных;
  • Оценка вреда, который может быть причинён Пользователю в случае нарушения законодательства о персональных данных;
  • Ознакомление работников Оператора с положениями законодательства о персональных данных.
